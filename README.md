  
## Visual Studio Code (VS code)
**Author: Milton Erick Otieno**
------------------------------------------------------------------------
Visual Studio code (VS code) is a free, lightweight and powerful code editor development by microsoft, It helps students write,run and debug programs efficiently.

-----------------------------------------------------------------------------
### Supported Progrramming languages
VS code supports many programming laguages such as:
1. Python
2. Javascript
3. HTML & CSS
4. Java
---------------------------------------------------
### Useful Extentions for students
Extensions improve functionality, Helpful ones include

1. **Python extention** - Run and debug python code
2. **Prittier**-Automaticaly format code
3. **Live server**- Preview web project live
4. **Gittlens**- Understand git changes
5. **Debugger**- Helps to detect errors
6. **Themes**- Improve appearance and readability

   ------------------------------------------------
   ### Overview of vs code features
   The following features are what makes Vs code a powerful and beginner-friendly

- *Built in terminal*-this feature helps your to run communds without leaving the editor
- *Smart code suggestions*- Suggest codes while you type, compleate words for your and helps to prevent errors.
- *Debugging tool*- This feature find and fixes mistakes (bugs) in your code.
